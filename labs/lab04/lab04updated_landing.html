This updated Lab 04 landing page documents the RSS 2.0 and Atom 1.0 feed work, including validation, formatting, feed content, and deployment checks.

Validation Requirements

  • Parse both XML files for well-formedness
  • Check RSS required elements: channel, title, link, description
  • Check Atom required elements: feed, title, id, updated, author
  • Validate against RSS 2.0 and Atom 1.0 rules

Content Requirements

  • At least 4 items/entries in each feed
  • RSS items include title, link, and description
  • Atom entries include title, id, updated, and summary/content
  • Entries link to actual completed lab pages

Workflow & Deployment Requirements

  • Use and maintain the `lab04` branch workflow
  • Commit progress incrementally as feed files are built
  • Deploy feed files and verify they are accessible on Azure
  • Include Lab 4 details and URLs in README documentation
