Lab 03: Personal Website

Home Labs Projects

Description

Build a multi-page personal website with consistent navigation, external styling, valid code, and a clear project/lab structure.

Core Requirements

  • Valid HTML5 on all pages with proper document structure
  • Valid external CSS and consistent formatting/indentation
  • Homepage, labs/projects listing page, and individual lab landing pages
  • Consistent header, navigation, content layout, and footer across pages
  • Working internal links and organized folder/file structure

Rubric Breakdown

  • Validity (10%): HTML and CSS pass W3C validation
  • Code Formatting (10%): readable indentation and organized external files
  • Site Structure (30%): required pages and consistent navigation
  • Design & Content (20%): cohesive visual design and meaningful content
  • Creativity & Quality (15%): branding, responsiveness, and enhancements
  • Workflow & Deployment (15%): Git workflow, Azure deployment, README

Verification Checklist

  • All pages load correctly from the deployed site
  • No broken internal navigation links
  • No inline CSS replacing external stylesheets
  • Repository shows branch/commit/PR workflow for Lab 03
  • README includes working URLs and a clear lab summary
