Lab 04: RSS & Atom Feeds

Home Labs Projects

Description

Create valid RSS 2.0 and Atom 1.0 XML feeds for your personal website with proper structure, complete content, and deploy them to the live Azure site.

Validation Requirements

  • Both XML feeds must be well-formed and parse correctly
  • RSS 2.0 required channel elements: title, link, description
  • Atom 1.0 required feed elements: title, id, updated, author
  • Feeds should comply with RSS 2.0 and Atom 1.0 specifications

Content Requirements

  • Minimum 4 items/entries in each feed
  • RSS items: title, link, description
  • Atom entries: title, id, updated, content or summary
  • Items should link to actual lab pages (Lab 1, Lab 2, Lab 3, and more if available)

Styling & Workflow Checks

  • Include XML stylesheet processing instruction (``)
  • Provide a CSS file for feed presentation (`rss.css` or equivalent)
  • Use a dedicated `lab04` branch with visible commit progression
  • Show PR workflow and include Lab 04 documentation in README

Live Verification

  • RSS feed URL is accessible on Azure and parses correctly
  • Atom feed URL is accessible on Azure and parses correctly
  • Linked pages resolve correctly from feed entries

Rubric Breakdown

  • XML Validity (15%)
  • Feed Structure (15%)
  • Content Completeness (25%)
  • Styling & Presentation (10%)
  • Workflow (35%)

Common Deductions To Avoid

  • Invalid XML syntax
  • Missing required RSS/Atom elements
  • Fewer than 4 items/entries
  • Copied example content without personalization
  • No `lab04` branch workflow or feeds not deployed
