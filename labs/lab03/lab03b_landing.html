Lab 03b: AI-Assisted Web Development Enhancement

Home Labs Projects

Description

Enhance the personal website using a combination of web development knowledge and AI collaboration. Focus on iterative prompting and measurable improvements in design, structure, and implementation quality.

Requirements

  • Create and work in a dedicated `lab3b` branch (do not merge to `main`)
  • Use AI tools to improve the current website
  • Complete 5 iterations, each driven by prompts of your choice
  • Commit after each iteration (minimum 5 commits)
  • Document how prompts were refined and how outputs changed per iteration

Iteration Focus

  • Prompt quality versus output quality
  • What changed after each prompt revision
  • How AI suggestions were validated and adapted
  • What was accepted, modified, or rejected and why

Deliverables

  • `lab3b` branch (not merged), submitted as a zip file
  • At least 5 commits representing the 5 iterations
  • A 2-page reflection `.docx` on AI-assisted development learnings
  • Place the reflection document in the `labs/lab03` folder for this branch

Submission Checklist

  • Branch history clearly shows iterative progress
  • Commits map to individual prompt/output improvements
  • Reflection explains workflow, tradeoffs, and lessons learned
  • All updated pages remain navigable and visually consistent
