# Lab 05: JavaScript Form Validation

## Description
Build an HTML form that uses external JavaScript to validate user input, provide feedback, and improve usability/accessibility (labels + focus styling).

## Files
- `labs/lab05/lab05_landing.html` (landing page + checklist)
- `labs/lab05/lab5.html` (form page)
- `labs/lab05/lab5.js` (validation + interactions)
- `labs/lab05/lab5.css` (styles; includes focus background color requirement)

## Rubric-style checklist (what graders look for)
- Name displayed under the heading
- Non-blank validation for all fields
- Success alert shown on valid submission
- Comments field clears/restores default text correctly
- Nickname button alerts: `"firstName lastName is nickname"`
- Labels properly associated to inputs (`for` + matching `id`)
- Focused fields have background color `#fee`
